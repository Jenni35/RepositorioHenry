Introducción

Esta es mi primera página web, y en ella hablaré sobre mi interés en aprender a programar.

Por qué quiero aprender a programar

Siempre me ha llamado la atención todo lo que se puede crear aprendiendo a programar. Desde páginas web hasta aplicaciones móviles, la programación abre un mundo de posibilidades. Me entusiasma la idea de poder desarrollar mis propias herramientas y contribuir a proyectos interesantes.

Mis Objetivos

  • Aprender HTML, CSS y JavaScript para crear páginas web interactivas.
  • Explorar otros lenguajes de programación como Python y Java.
  • Desarrollar proyectos personales para poner en práctica lo aprendido.
